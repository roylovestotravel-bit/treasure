# Treasure | Electric Bike Brand Website

一个从 0 开始搭建的网上商店演示项目,用于学习网页开发和 Git/GitHub 使用流程。

## 项目结构

```
store-project/
├── index.html   # 页面结构
├── styles.css   # 样式
├── script.js    # 购物车交互逻辑
└── README.md    # 项目说明
```

## 当前功能

- 商品展示(图文、价格、标签)
- 加入购物车 / 数量增减
- 购物车抽屉 + 实时计算总价
- 模拟结算(点击后仅提示,未接入真实支付)

## 本地预览

直接用浏览器打开 `index.html` 即可,不需要安装任何东西。

## 部署

本项目已托管在 GitHub 上,可通过 GitHub Pages 免费访问在线预览地址(纯前端演示,无真实下单功能)。
